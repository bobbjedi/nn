Подробное описание этого кода приведено в статье
Универсальная нейросеть на JavaScript

Демонстрация в браузере:
1) Открыть в Chrome файл index.html
2) Открыть консоль F12
3) Для обучения нейросети нужно подключить один из 4-х демонстрационных учебных наборов или СВОЙ
Набор для обучения записан в переменную trainingSet, набор для проверки в переменную fullCheckSet
Вызовите переменную trainingSet, чтобы проверить, что в учебном наборе.
4) Архитектуру нейросети требуется предварительно задать в переменную netSize
Функция startCheck() автоматически запускается при загрузке кода и проверяет число входов и выходов сети на соответствие учебным наборам
5) Запустить функцию checkAll(), которая проверяет все примеры из набора для проверки fullCheckSet и покажет ошибку сети
6) Запустить train(100), 100 эпох обучения нейрона на учебном наборе trainingSet
5) Повторно запустить checkAll()
